Contact Maageger

The Contact Manager application enables users to securely manage and organize their contacts through an intuitive web interface.

  • Backend:

    Spring Boot: Handles core backend logic, user authentication, email verification, and contact management.

    Spring Security: Manages authentication, OAuth2 setup, and authorization.

    JavaMailSender: Sends email verification links to new users during registration.

  • Frontend:

    Thymeleaf: Server-side rendering engine integrated with Spring Boot for templating and UI design.

    HTML/CSS and Taillwind: For styling and layout.

  • Database:

    MySQL: for contact storage, user data, and verification tokens.
